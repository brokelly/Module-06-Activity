Module 06 Activity - Final Documentation The purpose of the BG3 Builds website is to provide players of Baldur’s Gate 3 with an organized reference for different character builds and playstyles. The site allows users to explore various builds, view tier rankings, and save their favorite builds using web storage. The website was designed with usability and simplicity in mind so that players can quickly browse builds and filter them based on different criteria. Each build contains information about the character role, tier ranking, and a brief description of how the build functions within the game. The application uses HTML, CSS, and JavaScript to create an interactive experience. Flip cards are used to present the builds in a visually engaging way, while JavaScript manages filtering and storing favorite builds. Website Structure (Site Map) The BG3 Builds website contains three primary pages: Home (index.html) • Featured Builds • Favorite Builds Panel • What You'll Find Here • Build Roles Grid Builds Page (builds.html) • Build Filter Buttons • Tier Categories • Flip Cards for Builds About Page (about.html) • Project Overview • Website Purpose • Contact / Information Section Each page is connected through navigation buttons located in the header and footer of the website. Page Navigation and Button Functions Home Page (index.html) The Home page introduces the website and provides quick access to builds and features. Button User Action Result Home Click Returns user to the homepage Builds Click Opens the Builds page containing all character builds About Click Opens the About page with project details Featured Builds Users can view highlighted builds displayed as flip cards. Action Result Click a build card Card flips to reveal more information about the build Favorite Builds Panel Users can save builds they like. Action Result Add build to favorites Build is stored in browser Web Storage View favorites Displays saved builds in the favorites section Clear Favorites Removes all stored favorite builds Builds Page (builds.html) The Builds page contains the full list of character builds available on the website. Filter Buttons Users can filter builds based on tier rankings. Button Action Result All Click Displays all builds S Tier Click Shows only top tier builds A Tier Click Shows strong builds B Tier Click Shows situational builds Flip Cards Each build is displayed as a flip card. User Action Result Click card Card flips to show build details Click add to favorites Build is added to favorites list The cards automatically resize based on the amount of text displayed so that all information is visible. About Page (about.html) The About page explains the purpose and design of the website. Button Action Result Navigation buttons Click Moves between pages The page includes: • A description of the project • The purpose of the site • Information about how the site was built Technologies Used The BG3 Builds website was created using the following technologies: • HTML5 for website structure • CSS3 for styling and layout • JavaScript for interactivity and web storage functionality • Local Web Storage for saving favorite builds • Responsive design techniques for usability on different screen sizes GitHub Repository The project repository contains all source files used to build the application. Included files: • index.html • builds.html • about.html • styles.css • app.js • images folder • documentation The repository allows version control and provides a backup of the project files. The BG3 Builds website provides an interactive way for players to explore different character builds for Baldur’s Gate 3. By combining visual elements such as flip cards with JavaScript functionality like filtering and web storage, the application creates a simple but engaging user experience. The site structure and navigation allow users to easily browse builds, save favorites, and learn more about different character roles in the game.
